# CRM Integration (Salesforce)

This project integrates Salesforce CRM as a data source to enrich customer churn analysis.

## Data Used from CRM:
- Customer Accounts
- Opportunity (Sales Pipeline)
- Lead Sources

## Integration Steps:
1. Connected Salesforce CRM to Power BI using native connector
2. Imported CRM tables (Accounts, Opportunities)
3. Combined CRM data with churn dataset
4. Built dashboards for sales pipeline and risk analysis

## Business Value:
- Analyze impact of churn on revenue pipeline
- Identify high-risk customers in CRM
- Improve decision-making using combined analytics
